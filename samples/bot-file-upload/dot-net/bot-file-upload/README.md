# File Upload Bot - .NET (C#)

This sample demonstrates file upload capabilities for Microsoft Teams using .NET and ASP.NET Core.


## Interaction with bot
![bot-file-upload ](Images/bot-file-upload.gif)


## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)

## Run the sample

1. Navigate to this directory:
   ```bash
   cd csharp/bot file upload
   ```

2. Restore dependencies and run:
   ```bash
   dotnet run
   ```

The bot will start listening on `http://localhost:3978`.

Refer to the main [README.md](../../README.md) to interact with your bot in the agentsplayground or in Teams.
